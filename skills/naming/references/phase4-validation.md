# Phase 4 — Validation

Test what people infer, remember, pronounce, and trust. Keep creative judgment separate from
availability and legal checks.

## User Evidence

For each finalist:

1. Show the name with the same one-line product description for eight seconds.
2. Ask what the person expects from something with that name and record their first sentence
   verbatim.
3. After all candidates, ask which names they remember and note pronunciation or spelling stumbles.
4. Test with 5–8 people, mixing primary users with perceptive outsiders.
5. Compare reactions with the naming brief's intended feeling and tone.

Use stack ranking only after collecting unaided reactions. Look for useful polarization and clear
associations; universal indifference is a stronger warning than disagreement.

## Diligence Matrix

Record an explicit status for each check:

- **Trademark** — preliminary database scan by relevant class and jurisdiction; final status remains
  `needs counsel` until qualified legal review.
- **Language** — pronunciation, likely misspellings, homophones, slang, and native-speaker review in
  markets that matter.
- **Domain** — exact domain, sensible TLDs, modifiers, price, and confusion risks.
- **Handles** — availability on channels that matter to the business.
- **Competitive** — similarity in sound, spelling, meaning, category, and visual presentation.
- **Interface** — uppercase, lowercase, title case, headline, small icon, URL, and product-family use.
- **Future range** — extensions, demonym if relevant, and resistance to short-lived trends.

For a single-name gut check, combine this matrix with the Phase 3 dimensions and MMR test.

## Output

Produce a validation matrix with evidence, risk, and status for every finalist. Use `passed`,
`concern`, `failed`, or `not tested` rather than implying checks were performed.

Phase 4 is complete when 3–5 finalists have comparable evidence and explicit legal, linguistic,
domain, handle, competitive, interface, and user-test statuses. If fewer than three pass, return to
Exploration or Distillation.
